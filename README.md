## 📖 À propos

Ce projet est mon **portfolio personnel** développé avec **Java JEE** (Servlets & JSP). Il présente mes compétences, mes projets et mon parcours en tant qu'étudiant en Génie Logiciel et Ingénierie des Données.

**🔗 Lien vers le site en ligne :** [https://portfolio-kcb2.onrender.com](https://portfolio-kcb2.onrender.com)

---

## ✨ Fonctionnalités

- **Présentation personnelle** : Photo, description, informations de contact
- **Compétences** : Liste organisée par catégories (Langages, Frameworks, ML, etc.)
- **Projets** : Présentation de mes projets avec liens GitHub
- **Contact** : Liens vers email, WhatsApp, Facebook, LinkedIn
- **Responsive** : Design adapté à tous les écrans (mobile, tablette, desktop)
- **Téléchargement CV** : Bouton pour télécharger mon CV

### Frontend
